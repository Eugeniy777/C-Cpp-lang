В Eclipse есть встроенный отладчик. Это специализированное средство разработки, которое проверяет код на ошибки. Посмотрим, как оно работает.
Допустим, у вас есть программа, которая считает сумму чисел от 1 до n:

#include <iostream>
using namespace std;
int main() {
    int n = 5;
    int sum = 0;
    for (int i = 1; i <= n; ++i) {
        sum += i;
    }
    cout << sum;
    return 0;
} 

Проверив, что программа компилируется, запустите отладчик. Для этого нажмите на иконку с жуком. Отладчик предложит изменить текущее расположение окон на специальный набор окон для отладки. Согласитесь. Так будет удобнее просматривать значение переменной.
Пройдите программу по шагам, не заходя внутрь функций. Для этого есть опция “Step Over“. Чтобы запустить её, нажмите на иконку с закруглённой стрелкой. Выполнение переходит с пятой строчки на шестую, и значение переменной n выделяется жёлтым. Начинается цикл: меняются значения локальных переменных и переменная сумма. 
Когда сумма посчитана, завершите выполнение программы. Нажмите на иконку с зелёным треугольником. Программа отлажена. Иконка с надписью “C/C++“ вернёт вас в редактор кода.
Вы установили IDE и научились применять её основные инструменты. Это не раз пригодится вам при обучении и во время работы программистом. В следующей теме вернёмся к тренажёру и целочисленным типам С++.
