Установка Eclipse на Windows

Чтобы работать в Eclipse на Windows 64bit, понадобится компилятор MinGW64. Перейдите по ссылке и выберите MinGW-W64 Online Installer. Ссылка на скачивание компилятора.

Скачайте последнюю на текущий момент версию и запустите инсталлятор. После приветствия вы увидите окно настроек. Подойдёт такой вариант:

Далее выберите путь установки — без пробелов и кириллицы:

После установки проверьте работу компилятора. Из командной строки Windows зайдите в <путь установки mingw64>\mingw64\bin и выполните команду g++ --version:

Теперь пропишите путь в переменной среды Path (Панель Управления ⇒ Учётные записи пользователей ⇒ Изменение переменных среды). На скриншоте ниже — добавление значения в переменную Path под Windows 10:

Путь укажите до директории bin в переменной Path для вашего пользователя. Готово!
Теперь установите Eclipse на Windows 64bit. Eclipse написана на Java, поэтому вам понадобится виртуальная машина Java. Страница загрузки виртуальной машины Java.

Запустите инсталлятор и нажмите Install.

Скачайте саму IDE. Страница загрузки Eclipse с официального сайта.

Запустите инсталлятор и выберите Eclipse IDE for C/C++ Developers.
Укажите папку установки и нажмите Install.

После установки запустите Eclipse.
